# Emotion Recognition from Facial Images

Developed a deep learning-based facial emotion recognition system using **EfficientNetV2** to classify human emotions from facial images. The model was trained on a large-scale dataset containing **31,000+ images** spanning multiple emotion categories.

To address class imbalance in the dataset, techniques such as class weighting and data augmentation were employed, improving the model's ability to generalize across underrepresented classes. Extensive preprocessing and feature extraction were performed to enhance model performance.

The final model achieved **73% classification accuracy**, demonstrating the effectiveness of transfer learning with EfficientNetV2 for real-world emotion recognition tasks.

## Tech Stack

* Python
* TensorFlow / Keras
* EfficientNetV2
* NumPy
* Pandas
* Matplotlib

## Key Features

* Emotion classification from facial images
* Transfer learning using EfficientNetV2
* Handling of class imbalance through data augmentation and class weighting
* Training on a large-scale dataset of 31,000+ images
* Achieved 73% test accuracy
